About the Platform

Centralized Event Management for Students

UniEvent provides a clean and reliable interface for students to browse academic, social, technical, and sports events. The system is designed to support peak registration periods using cloud best practices.

AWS Deployment Design

Cloud Architecture Components

VPC

Custom network with public and private subnets across availability zones.

EC2

Application servers placed in private subnets for better security.

ALB

Distributes user traffic across healthy EC2 instances.

S3

Stores event posters, images, and related media securely.

IAM

Provides controlled access between EC2 and S3 without hardcoded keys.

Fault Tolerance

The system remains available even if one EC2 instance fails.
